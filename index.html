Part I — Design System & UI/UX Specification

Design Philosophy: The portal uses what I call institutional warmth — the authority of deep maroon and royal gold signals the gravity of the Office of the President, while clean white surfaces and icon-first navigation ensure a rural farmer in Monaragala with basic digital literacy can submit a report without confusion.

Color Palette: Deep Maroon #4A0A12 (primary authority), Royal Gold #C9A84C (accent, CTAs), Ivory #FAF8F2 (background — warmer than pure white, reduces eye strain on outdoor mobile use), Slate #2C3345 (body text — softer than black for extended reading).

Typography Stack: Display / Hero: Playfair Display (600, 700) — a serif with genuine gravitas that connotes statehood without feeling archaic. Body & UI: Inter (300–600) — the gold standard for screen legibility across all viewport sizes, excellent Sinhala/Tamil Unicode rendering fallback chain. Sinhala: Noto Sans Sinhala. Tamil: Noto Sans Tamil. These are loaded as secondary font faces and activated by the language switcher.

Accessibility for Rural Users: Navigation uses emoji icons + short text labels (never text alone), category selection uses 24px emoji touch targets meeting WCAG 2.1 AA, all form inputs are minimum 44px touch height, and the colour contrast ratio for all text on maroon exceeds 4.8:1.

// whatsapp-webhook.js
const express = require('express');
const axios = require('axios');
const app = express();

// Verify webhook signature (critical security step)
app.use((req, res, next) => {
  const signature = req.headers['x-hub-signature-256'];
  const payload = JSON.stringify(req.body);
  const expected = crypto.createHmac('sha256', process.env.WHATSAPP_APP_SECRET)
                         .update(payload).digest('hex');
  if (`sha256=${expected}` !== signature) {
    return res.sendStatus(403); // Reject spoofed webhooks
  }
  next();
});

app.post('/webhook/whatsapp', async (req, res) => {
  const { entry } = req.body;
  const message = entry[0]?.changes[0]?.value?.messages[0];
  if (!message) return res.sendStatus(200);
  
  const from = message.from; // Citizen's phone number
  let reportText = '';
  let imageUrl = null;
  
  if (message.type === 'text') {
    reportText = message.text.body;

// sms-parser.js — Parse SMS commands from basic phones
function parseSmsReport(smsBody, fromNumber) {
  const clean = smsBody.trim().toUpperCase();
  const keywords = ['ROAD', 'WATER', 'POWER', 'HEALTH', 'SCHOOL', 'FLOOD'];
  
  for (const kw of keywords) {
    if (clean.startsWith(kw + ' ')) {
      const rest = smsBody.slice(kw.length).trim();
      const parts = rest.split(' ');
      const district = parts[0];
      const village = parts[1];
      const description = parts.slice(2).join(' ');
      
      return {
        category: kw.toLowerCase() === 'power' ? 'electricity' : kw.toLowerCase(),
        district, village, description,
        phone: fromNumber,
        language: detectScript(smsBody) // Sinhala Unicode vs Latin
      };
    }
  }
  
  // Unstructured SMS — pass raw to AI for parsing
  return { text: smsBody, phone: fromNumber, unstructured: true };
}

AES-256 Encryption at field level (for PII):

# crypto.py — Field-level encryption for citizen PII
from cryptography.hazmat.primitives.ciphers.aead import AESGCM
import boto3, base64, os

def encrypt_pii(plaintext: str, district_id: str) -> str:
    """Each district has its own encryption key in AWS KMS"""
    kms = boto3.client('kms', region_name='ap-south-1')
    
    # Derive district-specific key from KMS master key
    data_key = kms.generate_data_key(
        KeyId=f"alias/district-{district_id}",
        KeySpec='AES_256'
    )
    
    aesgcm = AESGCM(data_key['Plaintext'])
    nonce = os.urandom(12)  # 96-bit nonce for GCM
    ciphertext = aesgcm.encrypt(nonce, plaintext.encode(), None)
    
    # Store: nonce + encrypted_key + ciphertext (all base64)
    return base64.b64encode(
        nonce + data_key['CiphertextBlob'] + ciphertext
    ).decode()

def anonymize_reporter(report: dict) -> dict:
    """Strip PII for minister-facing views — whistleblower protection"""
